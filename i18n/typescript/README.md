# Template: Hello World with 18n

Jovo Sample Voice App that uses the Jovo [i18next integration](https://www.jovo.tech/docs/output/i18n) with a simple "Hello World!" + asking for the user's name.

```sh
$ jovo new <directory> --template i18n
```
